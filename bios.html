BIOS/UEFI

This lesson covers the differences between BIOS and UEFI, as well as related components like EEPROM and CMOS.

BIOS vs. UEFI

The following table describes Unified Extensible Firmware Interface (UEFI), Basic Input/Output System (BIOS), Electrically Erasable Programmable Read-Only Memory (EEPROM), and Complementary Metal-Oxide Semiconductor (CMOS).

Component
Description
UEFI
Firmware that replaces BIOS, offers better security, faster boot times, and supports larger drives.
BIOS
Legacy firmware that controls startup and loads the OS into memory.
EEPROM
Non-volatile memory that replaced CMOS chips, allowing BIOS/UEFI updates.
CMOS
Legacy chip technology storing system settings, now replaced by EEPROM.

Troubleshooting and Updates

  • Check for BIOS/UEFI updates from the manufacturer.
  • Flashing the BIOS/UEFI can unlock new features and hardware support.
  • Ensure your PC is connected to a UPS before flashing to prevent power loss.
  • Access CMOS settings using a designated key (Delete, Insert, F1, F2) during startup.
