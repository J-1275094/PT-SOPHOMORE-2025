What is an Operating System (OS)?

An operating system (OS) is system software that manages computer hardware, software resources, and provides common services for computer programs. It serves as a bridge between the user and the computer hardware.

Proprietary vs. Open-Source Operating Systems

Proprietary OS: Created and maintained by a company. The source code is not shared with the public. Example: Windows, macOS.

Open-Source OS: Developed by communities or individuals. The source code is available for anyone to view, modify, and distribute. Example: Linux.
