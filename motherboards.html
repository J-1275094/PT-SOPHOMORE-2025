Motherboards

A motherboard (also called system board, logic board, or mainboard) is a circuit board that either houses, or connects to, all components operating in the computer.

Motherboard Specs

When selecting a motherboard, consider the following motherboard specifications:

  • Central Processing Unit (CPU) socket type
  • Memory module compatibility
  • Number of memory slots
  • Maximum supported memory
  • Expansion slot count and type
  • Onboard devices (video, audio, or network)

Typical Components

A typical motherboard includes the following components:

Component
Function / Characteristic
CPU socket
The CPU socket houses the CPU. Different types of motherboard sockets exist to fit different generations of CPUs. It's important that the socket is compatible with the new CPU's mounting holes and pin array.
Memory slots
Most motherboards have multiple slots to add memory cards. Memory slots are compatible with a specific type of memory module.
PCI
PCI slots are often called expansion slots. They allow you to add cards to expand the capabilities of a computer by installing expansion cards.
Integrated components
Many motherboards include integrated components such as network cards, audio cards, and video cards. They can also include USB and FireWire connections.

Troubleshooting Tips

  • Ensure your motherboard is compatible with the CPU socket type and memory module type.
  • Check that your motherboard has enough PCI slots for your expansion cards.
  • Verify that the onboard devices (video, audio, network) are functioning properly.
  • Make sure your motherboard's firmware (BIOS/UEFI) is updated to support your hardware.
