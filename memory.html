Memory

Random access memory (RAM) is important. The speed a PC can access memory contributes to its reliability and processing speed.

RAM Types

Type
Description
Static RAM (SRAM)
SRAM stores data using four transistors per bit. It is faster and requires less power than DRAM but has a lower storage capacity. Typically used in cache memory.
Dynamic RAM (DRAM)
DRAM stores data using a single transistor per bit and requires constant power. It has a high storage capacity and is used as the main system memory.

DRAM Standards

Standard
Description
DDR
First-generation DDR, operates at 2.5V, transfers two data sets per clock cycle.
DDR2
Doubles transfer rate of DDR, operates at 1.8V, has 240 pins.
DDR3
Doubles transfer rate of DDR2, operates at 1.5V, has 240 pins.
DDR4
Doubles transfer rate of DDR3, operates at 1.2V, has 288 pins.
DDR5
Doubles speed of DDR4, operates at 1.1V, has 288 pins with new architecture.

Memory Form Factors

Form
Description
DIMM
Has a 64-bit data path and is used in desktops. Different DDR generations have different pin counts.
SODIMM
Smaller than DIMM, used in laptops. Comes in 144-pin and 200-pin versions.
UniDIMM
Supports both DDR3 and DDR4, used in mobile platforms.

Error Correction Code (ECC) RAM

Type
Description
ECC RAM
Used in enterprise servers, detects and corrects data corruption. Has nine chips instead of eight.
